\doxysubsection{Detailed Description}
The \doxylink{class_character}{Character} class represents an entity in the game with various attributes and behaviors. 

Represents a character in the game with abilities, stats, and equipped items.

Characters have abilities, levels, stats, and can equip items to modify their attributes. This class provides the basic functionality for managing a character\textquotesingle{}s attributes and equipment.

This class manages the character\textquotesingle{}s attributes such as name, level, ability scores, and equipped items. It includes functionalities for modifying these attributes and calculating derived statistics.
