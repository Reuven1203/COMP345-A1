\doxysubsection{Detailed Description}
Class that can be observed by \doxylink{class_observer}{Observer} objects. 

The \doxylink{class_observable}{Observable} class represents objects that can be observed by \doxylink{class_observer}{Observer} instances. It maintains a list of observers and provides methods to add or remove observers. When the \doxylink{class_observable}{Observable} changes, it notifies all registered observers by calling their update method.
