\begin{DoxyAuthor}{Author}

\end{DoxyAuthor}
This Decorator will override the character\textquotesingle{}s accessor methods to reduce its attributes based on the stats of the item being unequipped. Will simultaneously set character\textquotesingle{}s bool for that Item\+Type (has\+Weapon(), has\+Helmet(), etc) to false.
