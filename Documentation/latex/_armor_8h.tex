Defines \doxylink{class_item_decorator}{Item\+Decorator} class, which allows items to be equipped by wrapping.

\end{DoxyAuthor}
This class allows for items to be equipped to characters, based on the type of item being equipped. This class should not be used on its own, rather instead using the subclasses (\doxylink{class_armor}{Armor}, \doxylink{class_helmet}{Helmet}, \doxylink{class_weapon}{Weapon}, etc). Holds a pointer to a character whose accessors are overridden in order to modify attribute values based on the items equipped/unequipped to/from the character.
