\doxysubsection{Detailed Description}
Concrete observer for displaying fighter state updates. 

\doxylink{class_fighter_observer}{Fighter\+Observer} is a concrete implementation of the \doxylink{class_observer}{Observer} class, designed to monitor changes in a \doxylink{class_fighter}{Fighter} object. When notified of a change, the \doxylink{class_fighter_observer}{Fighter\+Observer} will update its output to reflect the \doxylink{class_fighter}{Fighter}\textquotesingle{}s new state.
